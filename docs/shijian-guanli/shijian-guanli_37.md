# 七、解决拖延症

💡拖延症是现代人最普遍的负面情绪的来源，几乎每个人都会说我有拖延症。但这里一定要跟大家明确一下，拖延症并不是一种心理或者精神疾病，而是一种自我调节失败的结果。自诩有拖延症的人，会把计划要做的事情往后推迟，即便他们知道，那会让自己内疚，变得焦虑，任务也完成得更少。
