# 3.2 选品方向

因为美团的用户群体更偏年轻化，所以在兼顾实用性的基础上，选品更侧重于“颜值”，选品主要兼顾以下两点：

1）产品好看：年轻群体，颜值即正义，颜狗；

2）图片好看：主图好看，无太多水印文字，拍摄高端大气上档次。
