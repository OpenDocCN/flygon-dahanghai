# 第四步：获得授权

1）找对应持有品牌的三方进行商标授权（获取授权见「1.2.4 的品牌授权说明」）；

2）根据要入驻的店铺类型，授权书使用下方的固定模板即可保证一次过审；

3）一般为专营店，旗舰店需独占授权商标，价格高无流量倾斜，不建议选择。
