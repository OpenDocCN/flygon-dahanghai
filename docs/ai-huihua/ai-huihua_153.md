# 11.2.1 快速上手

一个快速的上手设计方法，公式：【LOGO 设计】+【LOGO 风格】+【你想要的画面】。

比如，我想为生财设计一个帆船图标 LOGO，用这样的方法，我可以快速设计出多组关键词。
