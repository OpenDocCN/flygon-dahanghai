# 3.2.2.4 避坑类笔记

•难度：2 颗星

•适用人群：适用于全部类目，好物带货和知识付费 IP 均可做。

避坑笔记可以是纯文字描述，也可以图文并茂，前者简单一些。内容架构有两种写法：

1.直接盘点总结大家最容易踩的坑、千万别做哪几种事；

2.先盘点总结雷区大坑，给出正确的做法。

•首图特点：纯色图/实拍图/网图都可以，图上要加标题花字突出主题。
