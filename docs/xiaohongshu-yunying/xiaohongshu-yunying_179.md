# 4）好物推荐型内容

•硬性要求：热爱买买买，喜欢分享，会拍照•软性要求：愿意出镜更佳•优点：制作笔记不难，完成起来比较容易，接广告很容易（也很容易置换产品）•缺点：出圈非常难！！赛道很卷！！！•变现方式：接广告（报备+非报备）
