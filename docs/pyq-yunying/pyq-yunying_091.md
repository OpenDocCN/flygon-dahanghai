# 3.5.3 第三种：故事型互动

也就是要读者在你的评论区写小作文。

这种互动要非常慎重。

我尝试过好多次，目前为止，最成功的就是要大家写作的七夕爱情故事与孝顺故事，其他的互动率不高。
