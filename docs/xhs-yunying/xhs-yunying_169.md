# 7.3 技巧三：话术引导 @杉木

用户添加我们的微信，通过好友之后（如果流量较大建议直接设置为加我好友时无需验证，否则容易显示添加好友频繁），欢迎语要固定下来，建议以问题开头，因为问句可以有效破冰，从而很自然地引出你要推荐的产品。

此外，提问还有个好处就是：先提问，战略制高点，变被动为主动，后面是顾客跟着你来走。
