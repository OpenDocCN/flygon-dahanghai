# 7.2.1 商品详情

千万不要做淘宝那种特别专业的美工图，详情图，主图最好都是小红书风格的图片，更有真实性。如果你是实拍的图或者视频，建议把详情图也都换成笔记里实拍的风格，让用户有”所见即所得“的感觉。
