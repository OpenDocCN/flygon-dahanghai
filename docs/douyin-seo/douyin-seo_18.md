# 3.2 核心关键词

问题：其他条件都不变，内容也一样的情况下，哪种号的排名优先级会更高？

选项 A：账号名称带上想要优化的关键词选项 B：账号名称不带想要优化的关键词

正确答案：A

带关键词的内容，更能够帮助平台识别和精准推荐。尤其是做直播间抢占搜索流量的时候，名称会起到比较大的作用。
