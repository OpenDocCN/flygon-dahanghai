# 7.4.1 开通视频号小店 @李诚

除了带别人的货，如果有条件，我们也可以开自己的「微信小商店」带自己的货，当自己的小店达到进入优选联盟的标准时，就可以把商品放在联盟里让达人带货了。

目前，视频号小店为 1 个号开 1 个店。首先我们先来了解一下开通一个小店都需要什么资质：
