# 7.4.3 如何入驻视频号橱窗售卖对应商品？@李诚

视频号橱窗保证金标准

•视频号橱窗基础保证金最低 100 元

•视频号橱窗浮动保证金（根据销售额标准提升）

•视频号橱窗商家、视频号橱窗达人在视频号橱窗中添加商品来源为微信小商店、小商店联盟、小程序联盟时需缴纳微信小商店管理保证金，缴纳标准为 30,000 元人民币。
