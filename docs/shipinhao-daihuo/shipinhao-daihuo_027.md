# 3.1.4 视频号选品推荐

书籍类：国学书籍、名人传记、婴幼儿绘本等；

服饰类：大码女装、头饰、脚链等；

家居百货类：房子防水材料、防水产品、面条机、老花眼镜、定时提醒的药盒、老人购物车、老人防走失用品、穿针器、拐杖、广场舞音响、老人收音机等；

其他：老人手机、多功能护理床。

这些都是微信视频号选品中的优质品类。

中老年人这个群体可以挖掘的空间很大。 中老年人和中老年人喜好风格也不太一样， 城市用户、农村用户的偏好也会不同。
