# 9.2 第二步：使用 ChatGPT 进行数据表内数据分析

将清洗后需要分析的数据表复制粘贴到 ChatGPT 中，进行分析。我们可以对 ChatGPT 进行提问，请 ChatGPT 给出数据分析结果。可以先让 ChatGPT 从它的角度给出数据表的分析角度，通过 ChatGPT 给出的反馈从各个角度去分析。

下面以使用 ChatGPT 分析人口数据的实际操作过程和具体步骤。

•Q: 你能分析一下数据表吗？

A：当然，你能提供你希望我分析的数据表吗？ 如果您还可以指定您希望我执行哪种类型的分析，那将很有帮助。

•粘贴数据表格到 ChatGPT 中

A：本数据表提供了 1949 年至 2021 年中国人口的信息。数据包括总人口、男女人口、城乡人口以及各组别占总人口的比例。

从数据中我们可以看出，中国的总人口逐年在稳步增长。 1949 年总人口 5416.7 万人，2021 年增加到 14126 万人。男性人口一直多于女性，男女人口比例保持相对稳定，除 2000 年代初男性人口比例略有增加。

在过去的几十年里，城市化一直是中国的一个重要趋势。数据显示，城镇人口占总人口的比重从 1949 年的 10.64%上升到 2021 年的 64.72%。相反，同期农村人口从 89.36%下降到 35.28%。

综上所述，数据显示了多年来中国人口和城市化的增长趋势。 男女人口比重保持相对稳定，城乡人口比重明显向城镇化转变。
