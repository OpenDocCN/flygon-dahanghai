
# (19赞)使用ChatGPT优化亚马逊listing流程，效率提升可观

作者：  阿路hh

日期：2023-06-02

 

 

下午一个做跨境电商的好友说想用ChatGPT结合亚马逊，优化一下写listing的流程。不确定可以达到什么效果，于是随手给我发了两条不同链接的listing，让GPT结合两个产品的功能输出一个新品的listing。

我在毫无调教的前提下直接让ChatGPT输出了一个新的listing，结果我觉得还挺满意的，毕竟以上的用时总共才5分钟不到。可想而知，如果调教一下，效率可以提升多少[机智]做跨境的朋友赶紧用起来！
