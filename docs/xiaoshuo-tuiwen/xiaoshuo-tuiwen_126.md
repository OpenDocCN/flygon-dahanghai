# 8.4 运营技巧三：抖音热点追踪 @小千

目前跟热点有两种方式：

•借助抖音热点宝等数据平台；

•刷自己账号的垂直度，关注足够多的对标账号后，通过刷“推荐”获取最新热点。

这里推荐第二种方法。因为热点宝里面收录的数据并不是很全面，有很多时效内的爆款作品都没抓取到。再加上抖音算法推荐机制优异，能够迅速推送和迭代热点。

具体操作逻辑如下：

首先，找到对标账号，参考「3.2.2 寻找对标账号」。在关注一批同类型账号后，去他们主页看作品和点赞。

一个账号只需要看主页的 1-2 个作品，且作品一定要从头看到尾。基本关注六七个对标账号后，就能在推荐里面刷到相关内容了。

刷到相关内容后，不要划走。耐心看完或者看一半，点赞评论不重要，最重要的是阅读时长。

当你把你的账号刷得足够垂直的时候，你才知道你究竟错过了多少跟爆款的机会。我们选爆款的时候，3 天前的作品就不考虑了，如果没有找到 3 天内的爆款，那只能说明一个问题，你刷得还是不够多。

最后提供一个参考标准：3 天内过 5000 点赞的都可以跟。

内容来源：3 月 3 日《从零开始，如何在抖音上做小说推文？》@小千
