# 9.3.5 找工作的「道」

前面写了术的层面的东西，再写几点关于找工作的「道」。

我觉得无论在 web 几，查理芒格给到的下面几条原则都是适用的：

1.不卖你不感兴趣的东西

2.不和你不喜欢的人一起工作

3.不为你不敬佩的人打工

如果不知道自己到底对什么感兴趣，可以先看自己对什么不感兴趣，然后排除；

或者，自己觉得打工就是个赚钱的工作，兴趣不重要，那就找自己擅长的，如果不知道自己擅长什么，那就找身边的人帮你判断你自己擅长什么。
